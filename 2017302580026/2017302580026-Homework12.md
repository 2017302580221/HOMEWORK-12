*c.* 由题意知，首部长度为5字节

存储转发时延 = (L\*8 + 5*8) / R，R = 622Mbps

当L = 1500时，存储转发时延 = (1500*8 + 40) / 622\*10<sup>6</sup> = 19μs

当L = 50时，存储转发时延 = （50*8 +40) / 622\*10<sup>6</sup> =0.7μs
